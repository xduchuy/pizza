description: Tự động kiểm tra thay đổi, viết commit message chuyên nghiệp và đẩy lên GitHub.
---
// turbo-all
1. Chạy lệnh `git status` để kiểm tra các file đã thay đổi.
2. Tự động tạo một nội dung thư sát (commit message) súc tích, chuyên nghiệp dựa trên những file bạn vừa thay đổi trong phiên làm việc.
3. Chạy lệnh `git add .` để gom toàn bộ file.
4. Chạy lệnh `git commit -m "nội dung commit bạn đã phân tích"` 
5. Chạy lệnh `git push origin main` để đẩy lên GitHub.
