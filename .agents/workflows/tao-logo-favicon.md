---
description: Tự động vẽ Logo hoặc Favicon dựa trên ý tưởng nội dung của dự án.
---
1. Hãy yêu cầu tôi mô tả ngắn gọn về tinh thần cốt lõi của thương hiệu hoặc dự án Web mà tôi đang làm.
2. Bạn tự động suy luận ra một câu Prompt hội họa chuyên nghiệp bằng Tiếng Anh (bao gồm miêu tả vector phẳng, phối màu hiện đại, tối giản).
3. Dùng công cụ `generate_image` để vẽ ra bức hình tuyệt đẹp đó.
4. Auto-lưu file hình ảnh đó vào gốc dự án của tôi với tên `favicon.png` hoặc `logo.png`
5. Tự động gắn thẻ `<link rel="icon">` vào file HTML.
6. Kết xuất hình ảnh ra Artefact để tôi chiêm ngưỡng!
